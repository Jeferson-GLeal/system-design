# System Design

Este repositório contém estudos e anotações sobre **System Design**, abordando a criação de sistemas distribuídos, escaláveis e resilientes.

System Design trata da **arquitetura de sistemas em larga escala**, considerando aspectos como desempenho, disponibilidade, tolerância a falhas e escalabilidade.

## Tópicos abordados

- Sistemas distribuídos
- Escalabilidade
- Alta disponibilidade
- Balanceamento de carga
- Estratégias de cache
- Arquiteturas orientadas a eventos
- Filas e mensageria
- Consistência de dados

## Conceitos importantes

- CAP Theorem
- Escalabilidade horizontal e vertical
- Comunicação entre microsserviços
- Processamento assíncrono
- Tolerância a falhas
- Observabilidade

## Tecnologias frequentemente utilizadas

Exemplos podem incluir:

- Docker
- Kubernetes
- Redis
- Kafka
- Nginx
- Infraestrutura em nuvem

## Propósito

Este repositório serve como **material de aprendizado e referência** para compreender como sistemas modernos de grande escala são projetados e operados.
